# Analyzing Arithmetic Intensity

For all of the following tasks, assume that:

1. A small number of rows and/or columns of any matrices can fit into cache.
2. Any vectors can fit into cache.
3. A small number of matrix blocks can fit into cache.

## Task 1

Estimate the arithmetic intensity of Vector-Vector element-wise multiplication.
In other words:

$c_i = a_i b_i$

Write out simple code or pseudo-code for a naive version of the operation, and show your math when evaluating arithmetic intensity.

## Task 2

Estimate the arithmetic intensity of Naive Matrix-Vector multiplication.
Write out simple code or pseudo-code for a naive version of the operation, and show your math when evaluating arithmetic intensity.

## Task 3

Estimate the arithmetic intensity of tiled Matrix-Vector Multiplication (tile the matrix much as you would in the case of Matrix-Matrix multiplication).
Write out simple code or pseudo-code for a naive version of the operation, and show your math when evaluating arithmetic intensity.

## Task 4

Compare the arithmetic intensity of the above operations with that of naive Matrix-Matrix multiplication and tiled Matrix-Matrix multiplication.
Describe any trends you observe.
