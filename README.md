# Analyzing Arithmetic Intensity

In other words:

$c_i = a_i b_i$

Write out simple code or pseudo-code for a naive version of the operation, and show your math when evaluating arithmetic intensity.

## Task 2

Estimate the arithmetic intensity of Naive Matrix-Vector multiplication.

## Task 3

Write out simple code or pseudo-code for a naive version of the operation, and show your math when evaluating arithmetic intensity.

## Task 4

Compare the arithmetic intensity of the above operations with that of naive Matrix-Matrix multiplication and tiled Matrix-Matrix multiplication.
